# Fontes TikTok Sans

Esta pasta deve conter os arquivos de fonte TikTok Sans.

## Como obter as fontes

1. Acesse o repositório oficial: https://github.com/tiktok/TikTokSans
2. Baixe os arquivos de fonte necessários:
   - `TikTokSans-Regular.woff2` (ou `.woff` / `.ttf`)
   - `TikTokSans-Medium.woff2` (ou `.woff` / `.ttf`)
   - `TikTokSans-Bold.woff2` (ou `.woff` / `.ttf`)

3. Coloque os arquivos nesta pasta (`public/fonts/`)

## Formatos suportados

O CSS está configurado para usar (em ordem de preferência):
- `.woff2` (melhor compressão)
- `.woff` (fallback)
- `.ttf` (fallback)

## Alternativa: Google Fonts ou CDN

Se preferir, você pode usar a fonte via CDN ou Google Fonts (se disponível) e atualizar o `fonts.css` conforme necessário.
